Lexicon Builder

Generating Lexicon will create a static Lexicon.generated.cs class (at the root of your project) with strong typed properties based on existing dictionary items.

Nested dictionary items are supported. Only leaf values of the dictionary tree are exposed as string properties of the Lexicon, and their parent nodes are represented by structs. The name of each string/struct is derived from its related dictionary item key (first parts of dot seperated keys are ignored, i.e. 'foo.bar.val' --> 'val').
